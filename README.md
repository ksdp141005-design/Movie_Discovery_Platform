# 🎬 Movie Discovery Platform

**Live Demo:** [Click here to view the live website](https://ksdp141005-design.github.io/Movie_Discovery_Platform/)

A fully responsive, dynamic web application that allows users to search for movies, series, and episodes using the OMDb API. Users can discover new titles, view details, and save their favorite movies directly to their local browser storage.

## 🚀 Features

- **Real-time Search:** Fetches and displays movie data instantly using the OMDb API.
- **Pagination:** Seamlessly navigate through multiple pages of search results.
- **Favorites System:** Save and remove favorite movies using Local Storage (data persists even if you close the browser).
- **Responsive Design:** Fully optimized UI for both desktop and mobile devices.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom styling, Flexbox, CSS Grid, and modern UI design.
- **JavaScript (ES6+):** DOM manipulation, asynchronous JavaScript (`async/await`), and API integration.
- **OMDb API:** External REST API for fetching movie data.

## ⚙️ How to Run Locally

1. Clone this repository: `git clone https://github.com/ksdp141005-design/Movie_Discovery_Platform.git`
2. Open `index.html` in your browser or use a VS Code extension like Live Server.
